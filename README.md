# Surya Sanity Portfolio

A high-end minimal landing page and portfolio architected with **Next.js 16**, **Tailwind CSS**, and **Framer Motion**.

### Core Specs
- **Identity**: Frontend Developer & UI/UX Designer.
- **Performance**: Static HTML Export (optimized for bare-metal Nginx hosting).
- **SEO**: Production-grade JSON-LD Person Schema and Metadata targeting *Surya Sanity* and *Surya Thangavel*.
- **Infrastructure**: Currently served from a bare-metal **POCO F1** server.
